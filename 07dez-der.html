Ziel: Die Welt komfortabel digital überfliegen.


Lösung: Flyover nutzen

Flyover auf dem iPhone oder iPad bietet gute Möglichkeiten, Vorträge, Erklärungen usw. anschaulich zu untermauern.

Wie geht man vor?

In der App „Karten“ können viele Großstädte und Sehenswürdigkeiten weltweit aus der Luft betrachtet werden. Flyover-Sehenswürdigkeiten sind an der Taste „Flyover“ auf den entsprechenden Ortskarten zu erkennen.
  1. Verwenden Sie die Taste oben rechts, um eine beliebige Karte (außer „ÖPNV“) auszuwählen.
  2. Tippen Sie auf den Namen einer Stadt oder auf den Namen einer Sehenswürdigkeit.
  3. Tippen Sie auf der Ortskarte auf „Flyover“. Wenn die Option „Flyover“ auf der Ortskarte einer Sehenswürdigkeit nicht angezeigt wird, tippen Sie auf die drei Punkte/mehr und dann auf „Flyover“.
  4. Führen Sie beliebige der folgenden Schritte aus:
    • Blickrichtung ändern: Richten Sie das iOS-Gerät in die gewünschte Blickrichtung aus oder neigen Sie es entsprechend.
    • Umgebung erkunden: Ziehen Sie mit dem Finger in eine beliebige Richtung. Legen Sie zum Ändern der Perspektive einen Finger auf den Bildschirm und drehen Sie einen zweiten Finger um den ersten.
    • 3D-Flug ausführen: Tippen Sie in der Karte unten auf dem Bildschirm auf „Tour starten“ oder „Stadttour starten“. Wenn die Karte nicht angezeigt wird, tippen Sie auf eine beliebige Stelle auf dem Bildschirm.
    • Tippen Sie auf das Kreuz, um zur Karte zurückzukehren. Wenn es nicht angezeigt wird, tippen Sie auf eine beliebige Stelle auf dem Bildschirm.
Eine Liste der Flyover-Orte finden Sie auf der Website zur Verfügbarkeit von iOS und iPadOS Features.
