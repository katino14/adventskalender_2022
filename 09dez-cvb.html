Kennen Sie Dobble? Das Spiel von Asmodee hat viele unterschiedliche Spielvarianten und ist schnell erklärt. Es macht einfach Spass. Es gibt Karten auf denen Symbole aufgedruckt sind und man muss das Symbol finden, das auf zwei Karten jeweils gleich ist. Klingt einfach? Ist es manchmal aber gar nicht, denn es ist immer nur eins gleich.

Jetzt hat @Profeesora sich die Mühe gemacht Sets zu erstellen, die im Fremdsprachenunterricht eingesetzt werden können. Es geht um den Grundwortschatz.

Quelle Bilder: @Profeesora

Das Set 1 besteht aus Symbolen zu Kleidung, Freizeit, Farben und Möbel. Hier gelangen Sie zur Download.

Das Set 2 besteht aus Symbolen mit Tieren, Lebensmittel, Zahlen und Schreibwaren.

Hier gelangen Sie zur Download.
